# 巨戟アーティア リセット回数カウンター

『モンスターハンターワイルズ』における  
**巨戟アーティア武器のスキル厳選（遺装置再付与）** において、  
消費ポイントから **リセット回数（1500ptsごと）** を自動計算・記録するためのブラウザツールです。

---

## 🔗 ツールURL
👉 https://sh854.github.io/Gog-weapons-reset-counter/
 
---

## ✍️ 使い方
1. 武器の **巨戟性能タイプ** を選択
2. 攻撃 / 会心 / 属性 の各遺装置について  
   - 開始時点の所持数（start）
   - 厳選後の所持数（now）
   を入力
3. 表示されたリセット回数を確認
4. 必要に応じて武器名を入力し「記録する」を押す
